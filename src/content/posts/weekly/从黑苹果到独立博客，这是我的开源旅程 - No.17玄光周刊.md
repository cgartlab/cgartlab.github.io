## 我的开源旅程

> 我的 GitHub 主页：[https://github.com/cgartlab](https://github.com/cgartlab)

我开始接触到的开源这个概念是在疫情初期，当时被关在家里百无聊赖，于是开始折腾起了黑苹果这个东西。所谓黑苹果就是在 Windows PC 上安装苹果系统。那个时候苹果还没有出 M 系列芯片，正版机器普遍都存在高价低配的问题，所以为了能同时用 Windows 和 Mac，开始了折腾。

当时国内讨论黑苹果的论坛很少，网站就那么几个。折腾过程不表，但这和开源有什么关系呢？其实 PC 之所以能装上苹果系统，本质是让苹果系统认为一台 Windows PC 是正常的苹果硬件，就需要对系统的引导文件进行一定的参数调整。包括不限于，各种驱动，硬件 ID 号，型号伪装等各种代码增减。

那时候我只是一个 " 索取者 "——在论坛和 GitHub 上找到别人分享好的 EFI 文件夹，下载、套用，能成功开机就算胜利。我很少去想这些文件是谁写的，又为什么要免费分享出来。

直到有一次，我遇到了一个毫无头绪的黑屏问题，只好硬着头皮去翻那些驱动和补丁的源码注释。在那些混乱的英文文档和代码提交记录里，我看到了几十上百人在为一个硬件的 ID 号反复调试、讨论。**那一刻我突然意识到：一群素不相识来自世界各地的人，仅仅是只为了一个共同的兴趣目标，就可以这样非营利地分工合作。** 这让我想起那个电驴还活着的 PC 互联网黄金时代，非常

那种感觉和单纯下载一个破解软件完全不同。我虽然只是改了改参数，但已经能感受到自己站在一个巨大的合作网络里。原来有那么多人遇到过和我一样的问题，这也让我开始思考：我能不能也从 " 拿来 " 变成 " 参与 "？

解决了大大小小十几个问题后，系统渐渐开始稳定下来。甚至，我开始把它用作搞定商单的主力机。稳定使用一段时间后，也尝试 [把我自己的引导文件上传到GitHub上了](https://github.com/cgartlab/Hackintosh---MSI-B360m-Mortar)。

## 本期推荐

### 基于数学曲线的动画合集

![](../_images/从黑苹果到独立博客，这是我的开源旅程%20-%20No.17玄光周刊-1777530794891.webp)

🔗 https://paidax01.github.io/math-curve-loaders/

这是一个专注于使用数学曲线创建精美加载动画的开源项目。展示了多种利用数学函数和几何原理实现的动态视觉效果，AI 给我总结了主要有这四大类：

- **贝塞尔曲线动画**：通过控制点生成的平滑曲线路径
- **三角函数动画**：利用正弦、余弦等函数创造周期性的流动效果
- **参数方程动画**：通过参数方程描绘的玫瑰线、心形线等特殊曲线
- **极坐标动画**：基于极坐标系生成的螺旋、花瓣等图案

每个动画都附带源代码，方便开发者学习和直接应用于项目中。理论上也是可以移植为 AE 表达式的，做 MG 动画的小伙伴可以玩玩，毕竟精度这么高的优雅动画很难用 K 帧来复刻。

### Krita - 十分低调的开源数字绘画工具

![SCP_03](../_images/从黑苹果到独立博客，这是我的开源旅程%20-%20No.17玄光周刊-1777530092204.webp)

🔗 https://krita.org

Krita 是我一直比较关注的开源绘画工具。它不像 Photoshop 那样是大杂烩，Krita 的定位非常清晰：专为数字艺术家服务的绘画应用。矢量图层、动画时间轴、丰富的画笔引擎，这些功能在商业工具里要花大价钱才能用到。Krita 的开发完全靠社区驱动，每次大版本更新都能看到用户反馈被认真对待。虽然新功能推出节奏较慢，但胜在十分稳定。

我曾经的 SCP 系列第三幅作品就是用 Krita 画的，很轻便，长边接近 8k 像素的画布在 16G 内存的 PC 上画起来也十分流畅。
