# CG艺术实验室

基于 Astro 构建的静态网站，专注于数字艺术、动态视觉设计、技术分享和知识管理。

## 技术栈

Astro + TypeScript + UnoCSS + MDX + KaTeX

## 项目结构

```text
├── public/icons/          # 网站图标
├── public/llms.txt        # LLM 站点索引
├── scripts/              # 构建脚本
├── src/components/      # 组件
├── src/content/posts/     # 博客文章
├── src/content/posts/weekly/  # 玄光周刊
├── src/i18n/            # 国际化
├── src/layouts/         # 布局
└── src/pages/           # 页面
```

## 主要功能

- 明暗主题、响应式布局、中英文双语
- llms.txt 自动生成（符合 llmstxt 规范）
- Giscus 评论、SEO 优化
